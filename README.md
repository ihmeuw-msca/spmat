# Special Matrices

A collection of tools for special matrices.
Currently include:

- `ILMat`: Identity plus PSD low rank matrix.
- `DLMat`: Diagonal plus PDS low rank matrix.
- `BDLMat`: Block `DLMat`.

## Install

To install, you could download this repository and,

```bash
pip install .
```

Or simply install through `pip`,

```bash
pip install spmat
```
